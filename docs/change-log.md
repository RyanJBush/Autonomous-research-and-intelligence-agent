# Europa Change Log

## 2026-05-12

### Retrieval-mode clarity
- Updated top-level project messaging to clearly distinguish:
  - deterministic static sample-data mode for repeatable demos, and
  - optional live API retrieval mode (Wikipedia OpenSearch + DuckDuckGo Instant Answer).
- Added explicit retrieval-mode guidance in README, screenshots docs, and preview copy.

### Limitations and disclaimers
- Reframed Europa as a student-built research assistant demo rather than a fact-checking system.
- Added direct language that outputs may be wrong (including hallucinations) and require independent human verification.
- Removed/avoided hype language and reliability overclaims.

### Preview and UX improvements
- Refined `docs/preview/index.html` with an icy blue/white visual style.
- Added visual pipeline section: Search → Retrieve → Synthesize → Output.
- Added feature cards, tech stack badges, repository link, and mobile-responsive layout.
- Added explicit disclaimer in preview: “Not a fact-checking service; verify results independently.”
