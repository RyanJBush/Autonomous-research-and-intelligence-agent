# Data

This directory holds sample and fixture data used by the docs, demos, and tests.

## `sample/`

- `sample_sources.json` — four mock sources covering a research question on enterprise AI governance. Used in docs and as an illustrative input for the agent.
- `sample_report.md` — a hand-crafted Markdown rendering of what an Astra AI report looks like for those sources.

These files are **mock data**. They do not represent real retrieval results and should not be cited. They exist so a reader can see the shape of the agent's I/O without running the full stack.
